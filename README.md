# ArtistStyleTransfer
 A model capable of learning an artist's style.

 Everything in the notebook is implemented on tensorflow.

 The original source image dataset is very large and it has a lot of images. The target dataset has only 10,200 images. The filenames in the source and target dataset are same.
 The extracted source images can be found in https://drive.google.com/drive/folders/16VGtYjvoqR8itfyBR_CIGpPaN7PIRo2G?usp=sharing.
